## 🗂️ Rotas da API

| Método | Endpoint       | Descrição               |
|--------|----------------|-------------------------|
| GET    | `/students`       | Listar todas as tarefas |
| POST   | `/students`       | Criar uma nova tarefa   |
| PUT    | `/students/{id}`  | Atualizar uma tarefa    |
| DELETE | `/students/{id}`  | Remover uma tarefa      |
